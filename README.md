# React Query Demo

A React application demonstrating advanced data fetching and management using React Query.

## Features
- Fetches posts from the JSONPlaceholder API
- Handles loading and error states
- Caches data using React Query
- Supports manual refetching of data

## Setup

Install dependencies:
npm install

Run the app:
npm run dev

## Technologies Used
- React
- React Query
- JSONPlaceholder API
